# Fixture provenance

Required by SPEC §6: every file in fixtures/ is recorded
here with its source, version, and license or public-domain status.
Synthetic fixtures are generated by a committed make_fixtures.py so they are
reproducible; real-data fixtures are fetched by script with local caching
and never committed above ~5 MB (no Git LFS).
